# 한국 주택 구매 완전 가이드 (2025-2026년 기준)

---

## 1. 자금 준비 단계

### 1-1. 총 필요 자금 산정

주택 구매 시 필요한 자금은 **매매가 + 부대비용(약 3~5%)**으로 구성됩니다.

| 항목 | 비율/금액 |
|------|-----------|
| 자기자본 (계약금 + 잔금 중 자기 부담분) | 최소 매매가의 20~30% |
| 대출 가능 금액 | LTV/DSR 기준에 따라 산정 |
| 부대비용 | 매매가의 약 3~5% (취득세, 중개수수료, 법무사비 등) |

### 1-2. 자기자본 확인 포인트

- 현재 보유 현금 및 예적금
- 전세보증금 회수 가능 금액
- 주식/펀드 등 유동화 가능 자산
- 가족 증여 (증여세 고려 필수: 직계존비속 5,000만원 / 배우자 6억원 비과세 한도)

## 2. 대출 관련

### 2-1. 핵심 용어: LTV, DTI, DSR

| 용어 | 의미 | 산식 |
|------|------|------|
| **LTV** (Loan to Value) | 담보인정비율. 주택 가치 대비 대출금 비율 | (주택담보대출 잔액 / 주택가격) x 100 |
| **DTI** (Debt to Income) | 총부채상환비율. 연소득 대비 주택담보대출 원리금 + 기타대출 이자 상환액 | (주담대 원리금 + 기타대출 이자) / 연소득 x 100 |
| **DSR** (Debt Service Ratio) | 총부채원리금상환비율. 연소득 대비 **모든 대출**의 원리금 상환액 | (모든 대출 원리금 합계) / 연소득 x 100 |

### 2-2. 2025-2026년 LTV 규제 현황

| 구분 | 규제지역 (투기과열지구/조정대상지역) | 비규제지역 |
|------|------|------|
| 무주택 세대주 | 50% (투기과열) / 60% (조정대상) | 70% |
| 생애최초 구입자 | 수도권/규제지역 70% | 80% |
| 1주택 세대 | 50% (투기과열) / 60% (조정대상) | 70% |
| 2주택 이상 | **0% (대출 불가)** (수도권/규제지역) | 60% |

### 2-3. DSR 규제 (스트레스 DSR 3단계)

| 구분 | 적용 기준 |
|------|-----------|
| 은행권 | DSR **40%** |
| 비은행권 (보험, 캐피탈 등) | DSR **50%** |
| 적용 대상 | 총 대출 1억원 초과 시 |

**스트레스 DSR 3단계** (2025년 7월 시행):
- 변동/혼합금리 대출 시 실제 금리에 **스트레스 금리 1.5%p**를 가산하여 DSR 산정
- 지방 주택담보대출은 2025년 말까지 0.75%p로 유예

### 2-4. 투기과열지구 고가주택 대출 제한

| 시가 구간 | 대출 한도 |
|-----------|-----------|
| 15억원 초과 ~ 25억원 이하 | **최대 4억원** |
| 25억원 초과 | **최대 2억원** |

### 2-5. 정책 대출 종류

#### (1) 내집마련 디딤돌대출

| 항목 | 내용 |
|------|------|
| 대상 | 무주택 세대주, 부부합산 연소득 6,000만원 이하 (신혼/2자녀 이상 8,500만원 이하) |
| 주택 가격 | 5억원 이하 (신혼/2자녀 이상 6억원 이하) |
| 면적 | 전용 85m2 이하 (읍면지역 100m2) |
| 대출 한도 | 최대 2.5억원 (신혼 4억원) |
| 금리 | 연 2.35%~3.95% |
| 만기 | 10/15/20/30년 |

#### (2) 보금자리론

| 항목 | 내용 |
|------|------|
| 대상 | 무주택/1주택 세대주, 부부합산 연소득 7,000만원 이하 (신혼 8,500만원, 다자녀 1억원) |
| 주택 가격 | 6억원 이하 |
| 대출 한도 | 최대 3.6억원 |
| 금리 | **고정금리** 연 3~4%대 |
| 특징 | 금리 변동 위험 없음 |

#### (3) 신생아 특례 디딤돌대출

| 항목 | 내용 |
|------|------|
| 대상 | 대출 신청일 기준 2년 이내 출생/입양 자녀가 있는 무주택 가구 |
| 소득 요건 | 부부합산 연소득 1.3억원 이하 |
| 자산 요건 | 순자산 4.69억원 이하 |
| 주택 가격 | 9억원 이하 |
| 대출 한도 | 최대 5억원 |
| 금리 | 연 1.6%~4.5% (특례금리 5년 기본, 추가 자녀 1명당 5년 연장, 최대 15년) |

#### (4) 시중은행 주택담보대출

- 금리: 변동/혼합/고정 선택 가능 (연 3.5~5.5%대)
- LTV/DSR 규제 내에서 한도 산정
- 중도상환수수료 존재 (보통 3년 이내 상환 시 부과)

### 5-2. 1세대 1주택 비과세 요건

모두 충족해야 함:
1. **1세대 1주택**: 세대 전원이 국내에 1주택만 보유
2. **보유 기간**: 취득일로부터 **2년 이상 보유**
3. **거주 요건**: 조정대상지역 내 취득 주택은 **2년 이상 실거주** 필수
4. **양도가액**: **12억원 이하** 시 전액 비과세, 12억원 초과 시 초과분에 대해서만 과세

| 일조량 | 남향 확인, 인접 건물 차폐 |
| 주차 | 세대당 주차 대수 |
| 관리비 | 최근 12개월 내역 확인 |

### 9-3. 주변 환경 확인

- 교통: 지하철역/버스 도보 거리
- 학군: 초중고 배정, 학원가
- 편의시설: 마트, 병원, 공원
- 혐오시설: 변전소, 소각장 인접 여부
- 개발 호재/악재: 재건축/재개발 계획, GTX 노선

## 10. 전체 플로우: 결심부터 입주까지

### STEP 1: 사전 준비 (1~3개월)

```
자금 점검 → 대출 사전 상담 → 청약통장 정리 → 희망 지역/조건 설정
```

- 자기자본 + 대출 가능 금액 + 부대비용 = **매수 가능 총액** 산정
- 은행 방문하여 **대출 사전 심사** (필수)
- 정책대출(디딤돌/보금자리론/신생아특례) 자격 확인

### STEP 2: 매물 탐색 (1~6개월)

```
실거래가 조사 → 매물 리스트업 → 현장 방문 → 비교 분석
```

- 국토교통부 실거래가 시스템으로 시세 파악
- 최소 **3~5곳 이상** 현장 방문
- 오전/오후/야간 각각 방문하여 일조량, 소음 확인

### STEP 3: 계약 체결

```
등기부등본 확인 → 매매 계약서 작성 → 계약금 지급 (매매가의 10%)
```

- 계약 직전 등기부등본 **당일 열람** 필수
- 계약금은 반드시 **매도인 본인 명의 계좌**로 송금

### STEP 4: 중도금 지급 (선택)

```
중도금 지급 (있는 경우) → 대출 본 심사 진행
```

### STEP 5: 잔금 및 소유권 이전 (계약 후 1~3개월)

```
잔금일 등기부등본 재확인 → 대출 실행 → 잔금 지급 → 소유권이전등기
```

- 잔금일 당일 아침에 등기부등본 **재열람**
- 잔금 지급 후 **60일 이내** 소유권이전등기 신청

### STEP 6: 신고 및 세금 납부

```
실거래 신고 (30일 이내) → 취득세 납부 (60일 이내) → 전입신고
```

### STEP 7: 입주 및 사후 관리

```
이사 → 전입신고 → 확정일자 → 하자 보수 청구 (해당 시)
```
